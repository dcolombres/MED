# Stack Tecnológico - MVP_med

## Backend
- **Lenguaje:** TypeScript / Node.js
- **Framework:** NestJS (Arquitectura Limpia)
- **Base de Datos:** PostgreSQL
- **ORM:** Prisma (recomendado por tipado y simplicidad)
- **Seguridad:** Validación con Zod (o validadores nativos de Nest), JWT para Auth, OWASP Best Practices.

## Frontend
- **Framework:** Next.js (App Router)
- **Estilos:** Tailwind CSS (Mobile First)
- **Lenguaje:** TypeScript
- **Estado:** React Context / TanStack Query (según necesidad)
- **UX/UI:** Google Stitch MCP para generación y prototipado.

## Infraestructura & Herramientas
- **Base de Datos:** PostgreSQL (Instalación local o Gestionada)
- **VCS:** GitHub (dcolombres)
- **Commits:** Conventional Commits
- **CLI:** gh (GitHub CLI)
